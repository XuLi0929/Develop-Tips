Preventing Data Loss: Automated Backups for Your Working Directory

Published: February 10, 2026 · 3-part guide · Updated: February 12, 2026

Part 2 of 3 — Add Automatic Scheduling

In the previous step, you created a script that can back up your directory on demand. That's useful, but you'll probably forget to run it. Let's make it automatic so you're always protected.

Modify the script to run in a loop with timestamps

Update your backup.sh to run every 60 seconds with a timestamp.

Replace the contents of backup.sh with:

#!/bin/bash
while true; do
    zip -r "backup_$(date +%Y%m%d_%H%M%S).zip" .
    sleep 60
done
